Codes folder:

There are four .py files and one .bin file in codes folder

They should be placed in the same folder.

To run the code to see the comparison of different search algorithm, you can run benchmark.py directly

octree.py or kdtree.py files can also be used for running for testing the correctness of the implementation.

result_set.py is a broker which defines the necessary data structures which can be used in tree implemtation. 

000000.bin is a specific point cloud data from Kitti dataset. You can also use other .bin files from the dataset to play with the algorithm. 

Documents folder:

There are two files in this folder. One is the summary and analysis of the results. The other is the
data for the plots.
